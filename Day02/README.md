# Northwind Database Analysis

## Overview

This project analyzes the Northwind SQLite database using SQL and Python (Pandas). The objective is to answer real-world business questions, visualize the results, and derive meaningful business insights.

## Dataset

- **Database:** Northwind SQLite Database
- **Source:** https://github.com/jpwhite3/northwind-SQLite3

## Tools Used

- SQL (SQLite)
- Python
- Pandas
- Matplotlib
- Google Colab
- GitHub

## Business Questions

The following business questions were answered using SQL:

1. What are the top 10 selling products?
2. Who are the top 10 customers by revenue?
3. What are the monthly sales trends?
4. Which product categories generate the highest revenue?
5. Which customers purchase most frequently?

## Project Files

- **queries.sql** – Contains all SQL queries used for the analysis.
- **analysis.ipynb** – SQL execution, data analysis, visualizations, and business insights.
- **northwind.db** – SQLite database used for analysis.
- **screenshots/** – Screenshots of SQL outputs and charts.

## Business Insights

- Louisiana Hot Spiced Okra is the top-selling product with 206,213 units sold.
- IT is the highest revenue-generating customer with a revenue of 9,745,371.29.
- Monthly sales show fluctuations over time, with peak sales occurring during the highest-performing month.
- Beverages is the best-performing product category, generating the highest revenue.
- IT placed the highest number of orders (335), indicating a strong and loyal customer relationship.

## Conclusion

This analysis demonstrates how SQL and Pandas can be used together to extract business insights from relational databases. By combining SQL queries with data visualization, valuable information about customer behavior, product performance, and sales trends can be identified.
